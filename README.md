Student Registration Form

A responsive and interactive Student Registration Form built using HTML, CSS and JavaScript. This project validates user input, displays error messages and ensures accurate form submission through JavaScript validation techniques.


Features:

- Full Name Validation
- Father Name Validation
- Seat Number Input
- Roll Number Input
- Gender Selection
- Department Selection
- Semester Selection
- Date of Birth Field
- Contact Number Validation
- Email Validation
- Address Field
- Password Validation
- Confirm Password Matching
- Error Messages for Invalid Inputs
- Responsive Design

Technologies Used:

- HTML5
- CSS3
- JavaScript (Vanilla JS)

Project Structure:


student-registration-form/
│
├── index.html
├── style.css
└── script.js


Project Objective:

The purpose of this project is to create a student registration system interface that validates user input and improves user experience through interactive feedback and form validation.

Concepts Used:

- Form Handling
- DOM Manipulation
- Event Handling
- Input Validation
- Conditional Statements
- Error Handling
- Responsive Design

Validation Features:

The form checks:

- Empty Fields
- Contact Number Length
- Password Length
- Password Confirmation
- Required Information

If validation fails, appropriate error messages are displayed to the user.
